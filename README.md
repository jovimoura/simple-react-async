# Simple React Async

It's a just project to study more about async with typescript and react

## Installation

### Clone

```
git clone https://github.com/jovimoura/simple-react-async
```

### Access the folder

```
cd <name-folder>
```

### Install dependencies

```
npm install
```

### Start the app

```
npm run dev
```

### Port:

```
http://localhost:5173/
```
